# 🌊 Dox - Data Professional's Advisor

Ask data-related questions and get AI-powered answers with PDF previews.

## Features
- 🤖 AI-powered answers to data questions
- 📄 Automatic PDF extraction and preview
- 🎨 Ocean-themed UI

## How to use
1. Type your question in the text box
2. Click "Submit"
3. If a PDF is referenced in the answer, a "Show PDF" button will appear
4. Click to view the first page of the PDF
